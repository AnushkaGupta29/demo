Required Format

b tag

In HTML, the 'b' tag is used to apply BOLD formatting to text. When you wrap text within b tags, the enclosed text will be displayed in a bold font style when rendered in a web browser

i tag

In HTML, the 'i' tag is used to apply ITALIC formatting to text. When you wrap text within i tags, the enclosed text will be displayed in an italic font style when rendered in a web browser.

u tag

In HTML, the 'u' tag is used to UNDERLINE text. When you wrap text within u tags, the enclosed text will be displayed with an underline when rendered in a web browser.

select tag

The 'select' tag in HTML is used to create a DROP-DOWN list, allowing users to select one or more options.

button tag

The 'button' tag in HTML is used to create a CLICKABLE button. You can place text, images, or other content inside a button , making it highly versatile.

hr tag

The 'hr' tag in HTML is used to insert a HORIZONTAL rule, which is a thematic break between sections of content.

br tag

The 'br' tag in HTML is used to insert a LINE BREAK, which means it creates a new line without starting a new paragraph.
